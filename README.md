## Setup

```sh
git clone https://github.com/GPT-Engineer-App/clean-professional-sms-dashboard.git
cd clean-professional-sms-dashboard
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
